Projects

Chatbot Application

As part of one of my larger second year modules, I was required to develop a Chatbot for the FDM website.

Our chatbot recognised language with spelling mistakes as well as small talk and FAQs for the FDM. What we achieved in a limited space of time included the creation of the chatbot itself as well as an admin system where admins can be created or deleted so they can edit the content of the chatbot and alter responses or add new questions that it can respond to. FDM were happy with our creation and I scored a first overall for this project and for my individual contribution and also for my skills at presenting the project at the end. This project combined my skills of using Spring Boot and Spring Security together with advanced Javascript, CSS and HTML. SQL Databases were used to store admin accounts and Dialogflow was our chosen API for help with the development of language understanding.

Task Managing Application

Another one of my second year modules required me to create a task managing application.

The idea was to create a command line program which efficiently managed the user's tasks along with their due date, name and description. You could mark when a task was finished, delete tasks, add new tasks and even save and load the tasks on launch of the program.

Although our finished project appeared to match many task managing applications, it was essentially a command line program as opposed to the standard GUI task managing programs out there.
